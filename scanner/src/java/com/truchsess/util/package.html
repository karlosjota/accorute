Provides classes and interfaces for handling trees of objects

The design is inspired by the java.util.Collections API. It gives access to the topology of a tree and allows to store elements of a given runtime type at each position. The nodes themself and their implementation-specific datamodel are not exposed.

The API is composed of

  • interfaces describing the various flavors of trees:
    • {@link com.truchsess.util.Tree}
    • {@link com.truchsess.util.SortedTree}
    • {@link com.truchsess.util.MutableTree}
    • {@link com.truchsess.util.ListTree}
    • {@link com.truchsess.util.MapTree}
    • {@link com.truchsess.util.MutableListTree}
    • {@link com.truchsess.util.MutableMapTree}
  • interfaces used to navigate through the tree:
    • {@link com.truchsess.util.TreeCursor}
    • {@link com.truchsess.util.SortedTreeCursor}
    • {@link com.truchsess.util.MutableTreeCursor}
    • {@link com.truchsess.util.ListTreeCursor}
    • {@link com.truchsess.util.MapTreeCursor}
    • {@link com.truchsess.util.MutableListTreeCursor}
    • {@link com.truchsess.util.MutableMapTreeCursor}
  • concrete implementation classes of tree:
    • {@link com.truchsess.util.ArrayListTree}
    • {@link com.truchsess.util.HashMapTree}
  • models that help building tree for UIs supporting 'selectable' or 'expandable/collapseable' nodes:
    • {@link com.truchsess.util.SelectableNode}
    • {@link com.truchsess.util.SelectableListTreeModel}
    • {@link com.truchsess.util.SelectableMapTreeModel}
    • {@link com.truchsess.util.ExpandableNode}
    • {@link com.truchsess.util.ExpandableListTreeModel}
    • {@link com.truchsess.util.ExpandableMapTreeModel}
  • and finally utility classes that allow to walk a tree:
    • {@link com.truchsess.util.TreeWalker}
    • {@link com.truchsess.util.MapTreeWalker}
