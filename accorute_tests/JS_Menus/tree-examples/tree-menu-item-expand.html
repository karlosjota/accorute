DHTML Tree Menu: XP Style Red
version 2.0

Samples:     Blue  |     Silver  |     Olive  |     Orange  |     Green  |     Red  |     Standard Tree  |


  • XP-Style
    You can create both standard or XP-like types of Tree Menu.
    This sample has XP-style.

  • Animation and Effects
    - Click on titles of submenus - submenus will collapse or expand.
    - Submenus have a fade effect in Internet Explorer when you collapse/expand them,
       so your menus look like original menus in Windows XP Explorer.
    - You can set different speed for animation (parameter: var tXPIterations)

  • Individual Styles
    Each item can have individual settings.
    You can create individual styles and assign them to items. See the sample.

    To create individual styles for standard items use var tstyles=[] array.
    To create individual styles for titles of submenus use var tXPStyles=[] array.
    For more details see Parameters Info.

    If you a novice in JavaScript or you want to save your time, use DHTML Tuner application.
    This software allows you to create and modify all our DHTML Menus easily and in no time.
