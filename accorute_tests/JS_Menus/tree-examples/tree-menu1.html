DHTML Tree Menu: Standard Tree Menu
version 2.0

Samples:     Blue  |     Silver  |     Olive  |     Orange  |     Green  |     Red  |     Standard Tree  |


  • Standard Style
    You can create both standard or XP-like types of Tree Menu.
    This sample has standard style.

  • Font, Colors, Borders, Images
    - Use any available font style;
    - Use any available border style and width;
    - Use any color for items background, menu background, font, and borders;
    - Assign images to icons, menu, items backgrounds.

  • Items Appearance
    Set custom height, cursor, alignment for items and icons.

  • Positioning
    Tree Menu can have absolute or static position (parameter: var tabsolute).
    It can be placed within a table/div.

  • Lines
    Assign your own images for lines.
