Single website licenses

You use the menu on one Internet or Intranet site (domain).
 
Who Is For Who don't have more than about one website (one domain) and want the best possible price.
 

How to Order

The secure order form has "Domain to register" field. Enter the domain where you'll use the menu.
For example your domain is "http://www.yourdomain.com/", then type "yourdomain.com" (w/o leading "http://www." and ending "/") in the "Domain to register" field.

If you want to use the menu on an Intranet you will need to enter the hostname of your website (as appears in the URL).
For example the link to your site is http://yourintranet/, then you should type "yourintranet" in the "Domain to register" field.
 
What You Get After you complete the payment via the secure form, you will receive a confirmation email within 12 hour. The email contains a keycode for domain you registered and further instructions.
 
How It Works The post-order email contains a keycode for domain you registered.
To register the menu you edit your data file (data.js by default) and add the key, for example:
var key="1234567890ab";
The key is valid throughout your website.
Within the website you can use the menu with the key as many times as you like.
W/o this key or on another domain the menu will show a nag message.
The key verification does not contact any outside servers to check your key.
 
Where to Use You can use the menu on an unlimited number of pages whose URL has the registered domain.
For example with a key for "yourdomain.com" you can place the menu on the following pages:
http://www.mycompany.com/home.html
https://www.mycompany.com/home.html
http://cgi.mycompany.com/more/file.html
http://mycompany.com/directory/back.html
etc.
