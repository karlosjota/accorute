DHTML Tabs is a DHTML+JavaScript product by Apycom Software.

Using DHTML Tabs you can make your website more powerful and attractive. Gentle tabs will refine the structure of your pages. Using DHTML Tabs is a very space-saving way to present the information. Now you don't have got to puzzle over the problem how to design the page in the best way.

If you have a large volume of data and your page needs to be scrolled down too much - DHTML Tabs is your choice! Just divide the information and distribute it among the tabs so that your visitors could reach the next piece of information with one single click. No more uncomfortable reading with scrolling!

Various types of data can be placed on one page. So, your visitors need no more reload a large number of pages to find the information they want.

DHTML Tabs will really lighten the process of site building as it allows to place much information within one page without creating many others.

But for all that, all the information is parted and structured. You are free to create tabs containing text, images and practically all kinds of information in various combinations. You may also create any desirable design inside the tab with HTML. And you can add as many tabs as you wish!


If you had an experience with other Apycom DHTML products, such as DHTML Menu & DHTML Tree Menu, you won't feel any difficulties working with DHTML Tabs. But even if not, installing, configurating and using DHTML Tabs will seem you as easy as ABC.

The tuning of DHTML Tabs is very easy.

All you need is to configure an existent data .js-file to fit your requirements.

Parameters have the following syntax:

  var parameterName = value;

The items' array has the following syntax:

  var bmenuItems = [
    ["text", "divID", "icon1", "icon2", "tip", "styleN"],
  ];

 Where:
    text - tab title (text="-" means item is separator);
    divID - ID attribute of object to show;
    icon1 - normal state icon;
    icon2 - mouse over state icon;
    tip - tab hint;
    styleN - individual style number.

For more details see the How to Setup info.

DHTML Tabs setup.

To setup DHTML Tabs paste the following code into the <head> tag of every page where you are going to have tabs:

<!-- Apycom DHTML Tabs -->
<noscript><a href=http://dhtml-menu.com/>DHTML Tabs, (c)2004 Apycom</a>
</noscript>
<script type="text/javascript" language="JavaScript1.2" src="apytabs.js"></script>
<!-- Apycom DHTML Tabs, dhtml-menu.com -->

 And then paste the following code into a place where you want the tabs to be shown:

<script type="text/javascript" language="JavaScript1.2" src="data.js"></script>

 Where:
  apytabs.js - the tabs engine file;
  data.js - your data file.

For more details see the How to Setup info.
